# 🤖 AI & Machine Learning Projects

4th Semester BS Artificial Intelligence assignments 
using real-world datasets.

## 📁 Projects

### Task 1 - Forest Fires Prediction (Supervised Learning)
- **Algorithm**: Random Forest Regressor
- **Dataset**: UCI Forest Fires (517 instances, 12 features)
- **Goal**: Predict burned area of forest fires
- **Techniques**: Label Encoding, MinMaxScaler, Log transformation
- **Metrics**: RMSE, MAE, R2 Score

### Task 2 - Air Quality Clustering (Unsupervised Learning)
- **Algorithm**: KMeans Clustering
- **Dataset**: UCI Air Quality (9471 instances)
- **Goal**: Cluster air quality patterns into groups
- **Techniques**: StandardScaler, Elbow Method, Silhouette Score
- **Result**: 3 clusters (Clean Air, Moderate, High Pollution)
